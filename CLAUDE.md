# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Commands

```bash
pnpm dev      # Next.js dev server (http://localhost:3000)
pnpm build    # Production build
pnpm start    # Serve production build
pnpm lint     # Next.js lint
```

No test runner or formatter configured. Package manager: pnpm (not yarn/npm).

## Stack

- Next.js 15 (App Router, RSC) + React 19 + TypeScript 5 (`strict: true`, `target: ES2022`, paths `@/*` → `./src/*`)
- Tailwind CSS v4 (PostCSS plugin). Theme tokens declared via `@theme` block in `src/app/globals.css`. No `tailwind.config.ts`. Config is CSS-first.
- shadcn-flavored UI primitives (Radix Slot + Label) hand-rolled in `src/components/ui/`. Not generated by the CLI; edit directly.
- Animations: `framer-motion` 11. Single reusable wrapper `<Reveal>` at `src/components/motion/reveal.tsx` (respects `prefers-reduced-motion`).
- Fonts: Geist Sans + Geist Mono via the `geist` package.
- Icons: `lucide-react`.
- Toasts: `sonner` (mounted in `RootLayout`).
- Mail: `nodemailer` SMTP.

## Architecture

Single-page portfolio at `/`, PT-BR copy. Theme: **Dracula** (official palette, dark only).

All sections composed in `src/app/page.tsx` (RSC).

- **Layout** (`src/app/layout.tsx`): defines `<html lang="pt-BR">`, applies Geist font CSS variables, sets full SEO `metadata` + `viewport`. Mounts `<Toaster>` once.
- **Theme** (`src/app/globals.css`): Tailwind v4 `@theme` block declares Dracula tokens (`--color-bg` #282a36, `--color-fg` #f8f8f2, `--color-purple` #bd93f9 accent, plus full Dracula palette `cyan/green/orange/pink/red/yellow`). Consumed via `bg-[--color-bg]` arbitrary classes throughout. To rebrand, change only this file.
- **Sections** (`src/components/sections/`): one file per section. Current order: `hero`, `about`, `experience`, `stack`, `projects`, `contact`. All wrap content in `<Reveal>` for scroll-triggered fade/blur/y.
- **Content** (`src/content/`): typed TS modules (`experience.ts`, `projects.ts`, `stack.ts`) are the single source of truth. No JSON, no CMS. `experience.ts` also exports `education`. Adding/editing experience/projects = edit these files.
- **Nav** (`src/components/nav.tsx`): sticky, switches to blur+border on scroll>24px. Anchors to section IDs (`sobre`, `experiencia`, `stack`, `projetos`, `contato`).
- **Footer** (`src/components/footer.tsx`): links to GitHub/LinkedIn/email.

### API routes (App Router)

- `POST /api/contact` (`src/app/api/contact/route.ts`): JSON body `{ name, email, message, about?, cellphone? }`. Validates required fields, returns 503 if SMTP env missing, 502 on send failure. Uses `runtime = "nodejs"` because nodemailer needs Node APIs.

### Env vars (`env-file.txt` is the template)

- `MAIL_AUTH_HOST`, `MAIL_AUTH_PORT`, `MAIL_AUTH_EMAIL`, `MAIL_AUTH_PASSWORD`: SMTP for contact form. Port 465 → `secure: true`, anything else → `secure: false` (handled in `src/lib/mail.ts`).

### SEO / social

- `src/app/icon.tsx`: favicon generated dynamically (Edge ImageResponse) as a 32×32 Dracula-themed "vs" monogram.
- `src/app/opengraph-image.tsx`: 1200×630 OG image generated dynamically with the Dracula palette.
- Title template `%s · Victor Souto` set in `layout.tsx`.

### Static assets (`public/`)

- `cv.pdf`: served at `/cv.pdf`, linked from Hero CTA. Currently the LinkedIn profile export.
- `avatar.jpeg`, `favicon.png`: legacy assets. Favicon comes from `icon.tsx`, not the PNG.

## Conventions

- All section components use PT-BR `id="..."` for nav anchors (`sobre`, `experiencia`, `stack`, `projetos`, `contato`).
- Use `cn(...)` from `@/lib/utils` for class merging. Never write `clsx` or `twMerge` directly.
- New colors/spacing/fonts go in the `@theme` block in `globals.css`, not in inline styles.
- Prefer server components. Mark `"use client"` only when needed (state, events, effects, framer-motion).
- Numbered section headers (`01 · Sobre`, `02 · Experiência`, …) via `<SectionHeader index="01" .../>`.
- Do not use em-dashes (`—`) in user-facing copy. Use `·`, `→`, `,`, `:`, or `.` instead.
